Phaser

Phaser es un framework de desarrollo para crear videojuegos 2D en HTML 5 (Software Libre), orientado a Pc y móviles, esta escrita sobre Javascript.

Phaser usa el canvas y WebGL para el renderizado y puede cambias entre ellos basado el soporte que tenga el navegador web.

Los juegos implementados en Phaser pueden ser compilador en iOS, Android y aplicaciones nativas para PC, a través de herramientas de terceros como Apache Cordova.
